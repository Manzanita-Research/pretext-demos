Why We Built This

Redaction tools produce dead documents. Static PDFs with black bars pasted over text, broken layouts, ugly gaps where information used to be.

We wanted a document that stays alive at every classification level — reflowing in real time, always looking typeset, always readable. Every word is a real DOM element: selectable, accessible, styled with CSS.

Pretext measures each paragraph's text so we can reserve stable heights. When you toggle classification levels, redacted spans swap between transparent text with black backgrounds and revealed text — but the layout stays locked because the space was pre-computed.
