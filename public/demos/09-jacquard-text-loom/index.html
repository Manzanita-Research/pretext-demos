The Jacquard Text Loom

Jacquard built a loom in 1804 that read punched cards to decide which threads to raise. Ada Lovelace looked at those cards and saw something else entirely — a general-purpose machine. Binary was born from silk. Now we run the circle back: binary weaving characters into cloth.
